class VNMRstream{....
This class simply allows the reading a writing of binary VNMR files (the binary FID files produced from Varian's INOVA, or similar system)...you can read and write 1D or 2D data file

IT will correct for any endian-ness (i.e. a SUN vs Linux vs Windows) binary types during the reading process...
Constructors
Function:
VNMRstream()
Input:
none
Description:
empty constructor--> Sets up the default VNMRstream
sets iomode=ios::in | ios::binary
Example Usage:
VNMRstream moo;
moo.open("infile"); //opens a file after declaration
Function:
VNMRstream(std::string infile, int iomode=ios::binary | ios::in)
IMPORTANT:
iomode can only be
ios::binary | ios::in
ios::binary | ios::out
Input:
infile-->the file you wish to begin extracting all the VNMRstream
iomode--> set the input/out mode
Description:
will either open a file for writing or reading....
Example Usage:
VNMRstream moo("myfile.txt", ios::binary | ios::in);
IO
Function:
bool read(Vector<complex> &fid)
bool read(matrix &fids)
IMPORTANT:
the iomode must be "ios::binary | ios::in"

Input:
varname--> the name of the variable within a matlab file...
var--> the variable you wish to fill from the VNMRstream file...the overloades you see aboave are the types allowed
Description:
--fills the 'var' from the data in the VNMRstream file...assuming that 'varname' exsists inside the file..
--If the file is a 2D file, you will only get the FIRST fid in the file for the Vector<complex> pieces
Example Usage:
VNMRstream inmat("fid", ios::binary | ios::in);
Vector<complex> vdat;
inmat.read(vdat);
Function:
bool write(Vector<complex> &fid)
bool write(matrix &fids)
IMPORTANT:
you can only write to the file once...there is no 'multiple' writings..only ONE FID per file...either a 1D (Vector) or 2D(matrix)
Input:
var--> the variable you wish to write to the file...
Description:
puts the data into a VNMR formated file..
Example Usage:
VNMRstream myout;
myout.open("fid");
Vector<complex> num(1024, 9.0);
myout.write(num);
Other Functions
Function:
void close()
Input:
none
Description:
closes the file...releases the file handle so you can manipulate it elsewhere if you deisre
Example Usage:
VNMRstream myout;
myout.open("fid");
Vector<complex> num;
myout.read(num);
myout.close();
Function:
bool is2D()
Input:
none
Description:
checks to see if the file is of 2D format...
Example Usage:
VNMRstream myout;
myout.open("fid");
if(myout.is2D()) cout<<"2D vnmr data file"<<endl;
Function:
bool is_open()
Input:
none
Description:
returns true if the file is open...false if not.
Example Usage:
VNMRstream myout;
myout.open("fid");
Vector<complex> num;
myout.read(num);
if(myout.is_open()) myout.close();
Function:
void open(const std::string fname, int iomode=ios::binary | ios::in)
Input:
fname--> file name to open...
iomode --> the mode for the opening...
Description:
reads the file into the object
Example Usage:
VNMRstream myout;
myout.open("fid");
Vector<complex> num;
myout.read(num);
