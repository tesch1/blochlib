class SpinSys:
public spin_sysPars{....
  • This class acts as a handy container for all the various bits of data associated with LISTS of atomic SpinSys data...(masses, atomic numbers, lables, etc) AND it also containes the Lists of spin operators...
  • This is the Master extension to both "SpinSys" and "MultiSpinOp" and should be used as the interface to them when dealing with 'liquid' like spins systems...
  • ***NOTE***::ALL The Spin operators ARE generated ONCE upon the the first call to the spin operator (or all are generated upon the 'GenSpinOps' call)...If you have ~15 spins all the generated matrices consume at LEAST (assumeing 15 spin 1/2) ~300Mb of memory...So BE CAREFUL...if you only want a 'few' spin operators uses the generators from the 'SpinOperators' section..see the example below
  • This is done because For solid state simulations most ALL the spin operators are used and used MANY times (due to powder averaging, and sample spinning etc)..to have all the spin-operators 'pre-generated' is a valuable speed enhancment.
  • Much of the Function Nameing was taken from 'Gamma':: this is simply to allow easy transistions..
Constructors
Function:
SpinSys()
Input:
void
Description:
empty constructor-->NO Spins...no nothing
Example Usage:
SpinSys moo;
Function:
SpinSys(int nsp)
Input:
nsp--> the number of spins in the system
Description:
Creates an SpinSys data set with 'nsp' spins.
Example Usage:
SpinSys moo(4);
int sys=moo.size(); //returns "4"
Function:
SpinSys(const SpinSys &cp)
Input:
cp--> an exsisting SpinSys to copy..
Description:
Creates a copy of the old SpinSys. It also COPIES the SPIN-OPS.
Example Usage:
SpinSys moo(3);
moo(2)="14N";
SpinSys loo(moo); //moo is copied into loo

Function:
double gamma(int which)
Input:
which--> The spin you wish to extract

Description:
Returns the gamma factor (gyromagnetic ratio) for the nucleous in units of Hz(rad)/Telsa. To get gamma factor in Hz you need to divide this number by 2Pi
Example Usage:
SpinSys moo(2);
cout<<moo.gamma(0); //prints out '26.7519E+7'
Function:
double gammaGauss(int which)
Input:
which--> The spin you wish to extract

Description:
Returns the gamma factor (gyromagnetic ratio) for the nucleous in units of Hz(rad)/Gauss. To get gamma factor in Hz you need to divide this number by 2Pi
Example Usage:
SpinSys moo(2);
cout<<moo.gammaGauss(1); //prints out '26.7519E+3'
Function:
int HS(int which)
Input:
which--> The spin you wish to extract

Description:
Returns the hilbert space dimension for the spin (HS=2*{spin quantum number}+1) FOR THE SPIN 'WHICH'
This number is essentially the array size needed to accuratly describe the spins behavior.
Example Usage:
SpinSys moo(4);
cout<<moo.HS(3); //prints out '2'
Function:
int HS()
Input:
none

Description:
Returns the hilbert space dimension for the spin (HS=2*<spin quantum number>+1)FOR THE TOTAL SPIN SYSTEM
The number is essentially the array size needed to accuratly describe the the collective spins behavior.

Example Usage:
SpinSys moo(4);
cout<<moo.HS(); //prints out '16'

Function:
Isotope &operator()(int which);
Isotope &operator[](int which);
Isotope operator()(int which) const;
Isotope operator[](int which)const;
Input:
which--> the isotope at position 'which'

Description:
This can be used to accsess any Isotope realted functions AND also for assignment of isotope types inside the SpinSys.
Example Usage:
SpinSys moo(2);
moo(0)="13C"; //set the first spin to be a 13C
moo[1]="1H"; //set the second spin to be a 1H
double 13cg=moo(0).gamma(); //the gamma factor for the first spin...

Description:
Determins weather or not the rhs is the same SpinSys as the lhs.
Example Usage:
SpinSys moo(2), loo(2);

Input:
in--> a string that would be a valid isotope label (i.e. 1H, 13C, etc)

Description:
Returns true IF 'in' is an isotope in the system and FALSE if not.
Example Usage:
SpinSys moo(2);
moo.isotopes("1H"); //true
moo(1)="13C";
moo.isotopes("14N"); //False
