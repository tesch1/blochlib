template<class ShapeEng_t>
class RotatingGrid :
public ShapeEng_t
Like the Gradient Grid, this is an extention to the Grid class, to allow for rotation of grids about one axis.
Constructors
Function:
RotatingGrid()
Input:
void
Description:
Default constructor. Set the default values
rate=0
axis=(0,0,1)
Example Usage:
RotatingGrid<XYZshape<> > grid;
Function:
RotatingGrid(const RotatingGrid &cp);
Input:
cp--> a RotatingGrid to copy
Description:
The copy constructor copies one Rotating Grid to the next. The ShapeEng_t must be the same for BOTH the copy and the copier.
Example Usage:
RotatingGrid<XYZshape<> > grid;
RotatingGrid<XYZshape<> > grid2(grid);

Function:
RotatingGrid(const ShapeEng_t &mygrid, const coord<> &axis=coord<>(0,0,0), const double &rate=0.0, bool continous=false ):
Input:
mygrid--> an XYZshape of some kind
axis--> the axis of grid rotation
rate--> the spinning rate
continous--> is the grid time dependant or is this a static rotation?
Description:
The main constructor. Sets the grid to the mygrid, the axis of rotation to 'axis' and the spinning rate to 'rate.' It also sets the 'continous' flag which if set to FALSE the grid will be rotated to the 'axis' but NOT be time dependant (i.e. not spinning); if it is set to TRUE then the grid assumes the grid points spin in time.
Example Usage:
Grid<UniformGrid> gg(mins, maxs, dims); //the basic grid
XYZfull tester; //the basic shape
XYZshape<XYZfull> jja(gg, tester); //the XYZshape
RotatingGrid<XYZshape<XYZfull> > jj(jja, myAxis, rate); //The Rotating grid

Function:
RotatingGrid(const coord<> &mins, const coord<> &maxs, const coord<int> &dims, const coord<> &axis=coord<>(0,0,0), const double &rate=0.0, bool continous=false ):

Input:
mins--> the minima for the ShapeEng_t
maxs--> the maxima for the ShapeEng_t
dims--> the dimensions for ShapeEng_t
axis--> the axis of grid rotation
rate--> the spinning rate
continous--> is the grid time dependant or is this a static rotation?
Description:
A main constructor, that allows you to set the grid parameters on the fly. Sets the axis of rotation to 'axis' and the spinning rate to 'rate.' It also sets the 'continous' flag which if set to FALSE the grid will be rotated to the 'axis' but NOT be time dependant (i.e. not spinning); if it is set to TRUE then the grid assumes the grid points spin in time. For use primarily with ShapeEng_t='XYZshape<XYZfull>'
Example Usage:
RotatingGrid<XYZshape<XYZfull> > jj(mins, maxs, dims, myAxis, rate); //The Rotating grid

Element Extraction
Function:
coord<> axis();
Input:
void
Description:
Retrieve the axis of rotation.
Example Usage:
Grid<UniformGrid> gg(mins, maxs, dims); //the basic grid
XYZfull tester; //the basic shape
XYZshape<XYZfull> jja(gg, tester); //the XYZshape
RotatingGrid<XYZshape<XYZfull> > jj(jja, myAxis, rate); //The Rotating grid
jj.axis(coord<>(1,1,1));
coord<> ax=jj.axis(); //will be [1/sqrt(3), 1/sqrt(3), 1/sqrt(3)]
Function:
coord<> operator()(int i, double t);
coord<> operator()(double t); --> FOR ITERATORS

Input:
i--> the grid point index
t--> a time
Description:
Retrieves the grid point at index i as a function of time t.
if 'continous' is false, this will return the rotated grid obtained upon construction.

Example Usage:
Grid<UniformGrid> gg(mins, maxs, dims); //the basic grid
XYZfull tester; //the basic shape
XYZshape<XYZfull> jja(gg, tester); //the XYZshape
RotatingGrid<XYZshape<XYZfull> > jj(jja, myAxis, rate); //The Rotating grid
jj.axis(coord<>(1,1,1));
coord<> myPt=jj(4, 0.0); //get the grid point 4 at t=0

Function:
double rate()
Input:
void
Description:
Returns the spinning rate.
Example Usage:
Grid<UniformGrid> gg(mins, maxs, dims); //the basic grid
XYZfull tester; //the basic shape
XYZshape<XYZfull> jja(gg, tester); //the XYZshape
RotatingGrid<XYZshape<XYZfull> > jj(jja, myAxis, rate); //The Rotating grid
